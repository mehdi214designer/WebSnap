Two ways to use WebSnap

A. From a live webpage

1. Install the WebSnap Chrome extension on any page

2. Click the WebSnap icon, pick viewport, hit Capture

3. Download the .wsnap file or copy to clipboard

4. Drop the file or paste here

B. From a local HTML file

1. Drop your .html file directly onto the import zone

2. Plugin renders it in a hidden iframe and extracts layout

3. Same import pipeline runs

HTML upload works best with self-contained files (inline CSS, data: URI images). External CSS/images may not load.
